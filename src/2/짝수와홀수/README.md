# 짝수와 홀수

## 문제
정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 
반환하는 함수, solution을 완성해주세요.

## 제한사항
- num은 int 범위의 정수입니다.
- 0은 짝수입니다.

## 입출력
|numbers|result|
|------|---|
|3|"Odd"|
|4|"Even"|

## 입출력 예 설명
```
없음
```

# 풀이 과정
1. 조건문을 이용해 num으로 받아오는 값을 2로 나누었을때 나머지가 0인 경우 "Even" 리턴 <br>
   (0은 짝수)
3. 0이 아닌경우 "Odd" 리턴
